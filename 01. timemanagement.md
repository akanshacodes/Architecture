# 2️⃣ TIME MANAGEMENT & DEEP FOCUS

The PDF starts with:

```text
Ek baar me ek kaam
```

Meaning:

```text
Undivided Attention
```

---

# 🔥 Why Important in IT?

Modern IT requires:
- Deep thinking
- Debugging
- Problem solving
- Learning tools
- Reading logs
- Troubleshooting

If your focus is divided:
- Learning becomes slow
- Mistakes increase
- Memory becomes weak

---

# 📌 Focus Rule

| Activity | Required Focus |
|---|---|
| Eating | Only eating |
| Class | Only class |
| Office | Only office work |
| Coding | Only coding |
| Learning | Only learning |
