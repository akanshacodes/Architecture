# 5️⃣ IAAS vs PAAS

VERY IMPORTANT CLOUD CONCEPT.

---

# 🌐 IAAS (Infrastructure as a Service)

Cloud provider gives:
- Virtual Machine
- Storage
- Networking

User manages:
- Operating System
- Applications
- Middleware

---

# 📌 IAAS Examples

| Cloud | Service |
|---|---|
| AWS | EC2 |
| Azure | Virtual Machine |
| GCP | Compute Engine |

---

# 🌐 PAAS (Platform as a Service)

Cloud provider manages:
- Infrastructure
- Runtime
- Scaling

User only deploys applications.

---

# 📌 PAAS Examples

| Cloud | Service |
|---|---|
| AWS | Elastic Beanstalk |
| Azure | Web Apps |
| GCP | App Engine |

---

# 🎯 Difference Between IAAS & PAAS

| IAAS | PAAS |
|---|---|
| More Control | Less Management |
| User Manages OS | Cloud Manages OS |
| Flexible | Easier Deployment |
| More Responsibility | Faster Development |
