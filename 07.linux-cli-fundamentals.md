- Mouse-based interaction
- Clicking buttons

```text

```

# 🎯 Important Understanding

- Case sensitive
- Syntax sensitive
- Space sensitive

---

# 9️⃣ LINUX COMMAND SYNTAX RULES

VERY IMPORTANT FOR BEGINNERS.

---

# 📌 Rules

## ✅ Commands must be accurate

Correct:
```bash
mkdir test
```

Wrong:
```bash
mk dir test
```

---

## ✅ Linux is Case Sensitive

```bash
test
```

and

```bash
Test
```

are different.

---

## ✅ Commands use spaces carefully

Correct:
```bash
cd foldername
```

---

### Purpose:

### Purpose:

# 🌐 cat

### Purpose:

# 1️⃣2️⃣ SHELL DEEP DIVE

MOST IMPORTANT LINUX CONCEPT.

---

# 🌐 What is a Shell?

Shell is:
```text
A translator between User and Operating System.
```

---

# 🌐 Responsibilities of Shell

Shell:
1. Takes command from user
2. Understands command
3. Sends instruction to OS
4. Returns output to user

---

# 🌐 Command Execution Flow

```text
User
   ↓
Shell
   ↓
Operating System
   ↓
Kernel
   ↓
Hardware
```

---

# 📌 Important Understanding

Without shell:
- Users cannot interact properly with Linux.
