# 4️⃣ ABSOLUTE PATH vs RELATIVE PATH

MOST IMPORTANT LINUX CONCEPT.

---

# 🌐 ABSOLUTE PATH

Absolute path:
```text
Always starts from the Root Directory (/)
```

---

# 📌 Example

```bash
cd /bin
```

---

# 🌐 Diagram

```text
/
├── config
├── chintu
├── chinki
└── munna
```

---

# 🎯 Key Understanding

No matter where you currently are,
absolute paths always begin with:
```text
/
```

---

# 🌐 RELATIVE PATH

Relative path:
```text
Starts from the Current Working Directory
```

---

# 📌 Example

```bash
cd ../munna
```

---

# 🌐 Meaning

```text
Move one directory back,
then enter the munna folder.
```

---

# 🌐 Relative Path Diagram

```text
/home/chintu/chinki
            ↑
      Current Location

cd ../munna
```

---

# 🎯 Important Point

Relative paths:
```text
Do NOT start with /
```

---

# 5️⃣ WHY RELATIVE PATHS ARE IMPORTANT IN AUTOMATION

VERY IMPORTANT DEVOPS CONCEPT.

```text
Automation me hmesha relative path use hota hai
```

---

# 🌐 Why?

Automation scripts should:
- Work dynamically
- Work across different environments
- Avoid hardcoded paths

---

# ❌ Hardcoded Absolute Path

```bash
cd /home/ubuntu/project
```

---

# ✅ Better Relative Path

```bash
cd ../project
```

---

# 🎯 DevOps Understanding

CI/CD pipelines and automation scripts commonly prefer:
```text
Relative Paths
```

because environments frequently change.

### ✅ Answer

---

### ✅ Answer

A path relative to the current working directory.

---

# ❓ What is Nginx?

### ✅ Answer

Nginx is a high-performance web server used for hosting websites and handling HTTP requests.

### ✅ Answer
