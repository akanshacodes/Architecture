# 6️⃣ GUI vs CLI

---

# 🖥️ GUI (Graphical User Interface)

GUI means:
- Mouse-based interaction
- Buttons and windows
- Click-based operations

Examples:
- Windows Desktop
- File Explorer

---

# ⌨️ CLI (Command Line Interface)

CLI means:
- Text-based interaction
- Commands execution
- Terminal usage

Examples:

```bash
pwd
ls
mkdir
```

---

# 🌐 GUI vs CLI Comparison

| GUI | CLI |
|---|---|
| Mouse based | Command based |
| Heavy resource usage | Lightweight |
| Beginner friendly | Automation friendly |
| Slower | Faster |

---

# 🎯 Why Production Uses CLI

Production servers prefer CLI because:
- Lower RAM usage
- Better performance
- Easier automation
- Better remote management

# 1️⃣4️⃣ IMPORTANT LINUX DIRECTORIES

---

# 📁 /bin

Stores basic Linux commands.

Examples:
- ls
- cp
- mv
- rm

---

# 📁 /boot

Stores boot files required during system startup.

---

# 📁 /etc

Stores:
- Configuration files
- Service configurations

VERY IMPORTANT FOR DEVOPS.

---

# 📁 /home

Stores user directories.

Example:

```text
/home/ubuntu
/home/rinki
```

---

# 📁 /lib

Stores system libraries required by applications.

---

# 📁 /media

Used for:
- USB devices
- Pendrive mounting

---

# 📁 /mnt

Temporary mount point for storage devices.

---

# 📁 /opt

Stores third-party software.

---

# 📁 /root

Home directory of:
```text
Root User (Administrator)
```

---

# 📁 /sbin

Stores administrative commands.

Examples:
- reboot
- shutdown

---

# 📁 /tmp

Stores temporary files.

---

# 📁 /var

VERY IMPORTANT.

Stores:
- Logs
- Cache
- Variable data

---

# 📌 Important DevOps Log Path

```text
/var/log
```

# 1️⃣5️⃣ RELATIVE PATH vs ABSOLUTE PATH

VERY IMPORTANT.

---

# 🌐 Relative Path

Path based on current location.

Example:

```bash
cd project
```

---

# 🌐 Absolute Path

Full path starting from:
```text
/
```

Example:

```bash
cd /home/ubuntu/project
```

---

# 🌐 Comparison

| Relative Path | Absolute Path |
|---|---|
| Current location dependent | Full path |
| No starting / | Starts with / |


---

# 🌐 ls

```bash

```

Lists files and folders.

---

# 🌐 ls -la

```bash

```

Shows:
- Hidden files
- Permissions

---

# 🌐 mkdir

```bash
mkdir project
```

Creates folder.

---

# 🌐 touch

```bash
touch file.txt
```

Creates empty file.

---

# 🌐 cd

```bash
cd foldername
```

Changes directory.

---

# 🌐 cp

```bash
cp file1.txt backup.txt
```

Copies file.

---

# 🌐 mv

```bash
mv old.txt new.txt
```

Moves or renames file.

---

# 🌐 rm

```bash
rm file.txt
```

Deletes file.

---

# 🌐 rm -rf

```bash
rm -rf foldername
```

Deletes folder recursively.

⚠️ Dangerous command.
