# 🔟 HOW WEBSITES WORK INTERNALLY

---

# 🌐 Step-by-Step Website Working

---

## STEP 1

User opens browser.

---

## STEP 2

User enters URL.

Example:

```text
https://netflix.com
```

---

## STEP 3

Browser sends request to server.

---

## STEP 4

Web server receives request.

---

## STEP 5

Server finds website files.

---

## STEP 6

Server sends response.

---

## STEP 7

Browser renders website.

---

# 🔥 Complete Flow

```text
Browser
   ↓ Request
Web Server
   ↓ Response
Browser
   ↓
Website Visible
```

# 1️⃣1️⃣ BROWSER REQUEST LIFECYCLE

This is extremely important.

---

# 🌐 Lifecycle

```text
1. User enters URL
2. DNS resolves domain
3. Browser contacts server
4. Server processes request
5. Response generated
6. Browser renders page
```

---

# 📌 Example

```text
youtube.com
```

Browser first finds:
```text
Server IP Address
```

Then contacts server.

---

# 2️⃣5️⃣ IMPORTANT INTERVIEW QUESTIONS

---

# ❓ What is a Web Server?

### ✅ Answer

A web server is software that listens to browser requests and serves webpages using HTTP/HTTPS protocols.

---

# ❓ What is localhost?

### ✅ Answer

localhost refers to the local machine itself and uses IP address 127.0.0.1.

---

# ❓ What is IIS?

### ✅ Answer

IIS is Microsoft's web server software used to host websites on Windows systems.

---

# ❓ Difference Between Client and Server?

| Client | Server |
|---|---|
| Sends Request | Sends Response |
| Browser | Web Server |

---

# ❓ What is Deployment?

### ✅ Answer

Deployment is the process of making an application available on servers for users.
