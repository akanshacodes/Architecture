# 🔟 HOW A WEBSITE WORKS

---

# 🌐 Website Working Flow

---

## STEP 1

User opens browser.

---

## STEP 2

User enters URL.

Example:

```text
https://netflix.com
```

---

## STEP 3

Browser sends request to web server.

---

## STEP 4

---

## STEP 5

---

## STEP 6

Server sends response.

---

## STEP 7

Browser renders webpage.

---

# 🔥 Complete Flow

```text
Browser
   ↓ Request
Web Server
   ↓ Response
Browser
   ↓
Website Visible
```

# 1️⃣1️⃣ BROWSER REQUEST LIFECYCLE

This is an extremely important concept.

---

# 🌐 Request Lifecycle

```text
1. User enters URL
2. DNS resolves domain
3. Browser contacts server
4. Server processes request
5. Response is generated
6. Browser renders webpage
```

---

# 📌 Example

```text
youtube.com
```

The browser first finds:
```text
Server IP Address
```

Then communicates with the server.

---

MOST IMPORTANT FOR WINDOWS SERVERS.

---

# 🌐 IIS Definition

IIS is Microsoft's web server software used for hosting websites on Windows systems.

---

# 📌 Full Form

```text
Internet Information Services
```

---

# 📌 Responsibilities of IIS

- Host websites
- Receive browser requests
- Serve webpages
- Manage local web applications

---

# 📌 Default IIS Website Path

```text
C:/inetpub/wwwroot
```

---

# 📌 Meaning

Any website files placed inside:
```text
wwwroot
```

can be served by IIS.

### ✅ Answer

### ✅ Answer
